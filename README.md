# Turf சென்னை - Premium Sports Turf Booking Website

A production-ready marketing website for Turf சென்னை, Chennai's premier sports turf booking platform. Built with bold sports energy aesthetic featuring dark brutalist design with neon athletic accents.

## 🏟️ Features

- **Instant Booking System** - Live slot calendar with WhatsApp confirmation
- **Premium Facilities Showcase** - Photo grid of turf quality, floodlights, and equipment
- **Transparent Pricing** - Clear hourly rates for cricket nets, football, and full pitch
- **Interactive Location Map** - Embedded Google Maps with directions
- **Social Proof** - Customer testimonials and action shots
- **Mobile-First Design** - Responsive across all devices with sticky mobile CTA
- **Smooth Animations** - Scroll-reveal effects and hover interactions

## 🎨 Design System

### Color Palette
- **Primary**: #0FFF50 (Electric neon green - turf grass energy)
- **Secondary**: #FF3D00 (Deep orange - cricket ball energy)
- **Accent**: #00E5FF (Electric cyan - hover states)
- **Background**: #0A0A0A (Near-black - night match atmosphere)
- **Surface**: #1A1A1A (Dark card backgrounds)
- **Text**: #F5F5F5 (High contrast)

### Typography
- **Headings**: Bebas Neue (Bold, condensed, sports poster energy)
- **Body**: Inter (Clean, readable, UI-optimized)
- **Tamil**: Noto Sans Tamil (For Tamil script)

## 🚀 Tech Stack

- **Framework**: Vite + React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel-ready

## 🎨 Design Philosophy

**Bold Sports Energy** - Dark brutalist base meets neon athletic energy. Stadium floodlights cutting through night, electric green turf lines, high-contrast action photography. Asymmetric layouts, strong geometric sans typography, zero corporate softness.

### Anti-Slop Features
- ✅ Specific, benefit-led copy (no "innovative solutions")
- ✅ Strong typographic hierarchy with generous spacing
- ✅ Restrained color usage (primary accent only for CTAs and highlights)
- ✅ Real imagery from Unsplash (turf-specific action shots)
- ✅ Subtle depth with borders, shadows, and rounded corners
- ✅ Micro-interactions on hover and scroll
- ✅ Semantic HTML with proper accessibility

## 📱 Mobile Optimization

- Fixed header with mobile hamburger menu
- Stacked layout for all sections
- Touch-friendly buttons (44px minimum)
- Floating WhatsApp button for instant contact
- Smooth scroll navigation
